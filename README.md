# Adam's Spotify Website

A simple website that shows off my Spotify playlists so friends and family can give me musical tips. Each playlist is an official Spotify player, so it always shows every track and stays up to date — no manual updating of songs.

## How to add or change a playlist (no coding needed)

1. Open the file **`playlists.js`**.
2. In Spotify, open a playlist → click **···** → **Share** → **Copy link to playlist**.
3. Paste that link into `playlists.js` following the example already in there.
4. Save the file. Done.

## How to see it

- **On your computer:** double-click `index.html`.
- **On the web:** this repo is set up for free hosting with GitHub Pages (see below).

## Files

| File | What it is |
|------|------------|
| `playlists.js` | **The only file you edit** — your list of playlists. |
| `index.html` | The page itself. |
| `styles.css` | How it looks. |
| `app.js` | The behind-the-scenes glue. |
