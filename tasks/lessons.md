# Lessons

## Don't sanitize the user's own content without asking
- **Mistake:** Silently softened the playlist title "Better than the shit you usually listen to" to "...Stuff..." because I assumed a family-friendly audience.
- **Pattern:** When reproducing the user's own words/data (playlist names, copy, etc.), preserve them verbatim. If tone/appropriateness seems like it *might* matter, ask — don't unilaterally edit.
- **Rule:** Faithful reproduction by default. Editorial judgement only on request.
